# react-twelve

一个用于紫微斗数十二宫计算的 React 组件。

## 安装

```bash
npm install react-twelve
# 或
yarn add react-twelve
```

## 使用

```jsx
import React from 'react';
import Twelve from 'react-twelve';

function App() {
  return (
    <div>
      <Twelve />
    </div>
  );
}

export default App;
```

## 依赖要求

- React >= 16.8.0
- React DOM >= 16.8.0
- Ant Design >= 5.0.0
